# ホーム画面 画面イベント設計

## 1. 概要
本画面はアプリケーションの玄関口であり、主要な機能への入口を提供する。背景にはダイナーの外観画像（ming_outlook.png）を配し、左側にはシステム名称を表示するスタイリッシュなデザインとする。

## 2. 画面要素
| 要素名 | 種類 | 説明 |
| :--- | :--- | :--- |
| ロゴアイコン | 画像 | ming_10th_icon.png。クリックでホームへ。 |
| システム名称 | テキスト | 「料理原価計算システム」。中央左に配置。 |
| ナビゲーションタブ | リンク | 各機能画面への遷移を行う。 |
| 背景画像 | 画像 | ming_outlook.png。右側に配置。 |

## 3. 画面イベント
### 3.1 ページロード
- ユーザー認証状態の確認。
- ホーム画面のレイアウトを表示。
- 背景画像とタイトルにフェードインアニメーションを適用。

### 3.2 ナビゲーションクリック
- 以下の各項目をクリックした際、対応するURLへ遷移する。
  - **ホーム (HOME)**: `/`
  - **食材追加 (ADD)**: `/ingredients/add`
  - **食材編集 (EDIT)**: `/ingredients/edit`
  - **食材検索 (SEARCH)**: `/ingredients/search`
  - **中料理 (MEDIUM)**: `/dishes/medium`
  - **大料理 (LARGE)**: `/dishes/large`
  - **販売価格計算 (CALCULATOR)**: `/calculator`
- アクティブなタブは太字（font-bold）かつ黒色（text-black）で強調表示される。

### 3.3 ホバーアクション
- ナビゲーション項目にマウスオーバーした際、背景色を微かに明るく（hover:bg-white/50）変化させる。
