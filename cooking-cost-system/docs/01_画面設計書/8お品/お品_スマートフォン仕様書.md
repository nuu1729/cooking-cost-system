# お品 スマートフォン仕様書

## 1. 概要

お品（メニュー）を作成する画面。仕込みと同様に左右2カラム構成を上下縦1カラムに変更する。

| 項目 | 内容 |
|---|---|
| 対象ブレークポイント | `max-width: 640px` |
| デスクトップとの差異 | 左右2カラム → 上下1カラム |

---

## 2. 画面要素

デスクトップ版と同一要素。配置のみ変更。

---

## 4. デスクトップとの変更点

| 項目 | デスクトップ | スマートフォン |
|---|---|---|
| レイアウト | 左右2カラム | 上下縦1カラム |
| 仕込み選択エリア | 左カラム | 上部セクション |
| 構成リスト | 右カラム上部 | 中央セクション |
| 確定エリア | 右カラム下部 | 下部セクション |
