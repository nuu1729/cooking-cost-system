# お品 画面仕様書

## 1. 概要
「お品」画面は、登録済みの「仕込み品」を組み合わせて、最終的な提供メニューである「お品」を作成・構成するための画面である。
仕込みページと同様のレイアウトやデザインを基準とし、直感的な操作で原価を自動算出し管理することができる。

## 2. 画面要素
### 2.1 左カラム：検索・追加エリア
登録済みの仕込み品（属性が「仕込み品」である『2』のもの）を検索し、使用量を入力してリストに追加するエリア。

| 要素名 | 種類 | 説明 | 備考 |
| :--- | :--- | :--- | :--- |
| **仕込み（検索）** | テキスト入力 | 入力内容に基づきDBの仕込み品（属性「2」）を検索し、サジェストからユーザーが選択する。 | 初期：空 |
| **量** | 数値入力 | 使用する量を半角数字で入力。 | 初期：0 |
| **単位** | 表示項目 | 仕込み品は単位が「g」固定であるため、「g」を表示する。 | 初期：g（変更不可） |
| **値段** | 表示項目 | 選択された仕込み品の単価とユーザーが入力した量から算出結果を表示。 | 初期：¥ 0 |
| **リストに追加** | ボタン | 入力内容を確定し、右側のリストへ追加する。 | （仕込み画面の「仕込みに追加」相当） |

### 2.2 右カラム：リスト・確定エリア
追加された仕込み品の一覧と、お品名の設定、および確定操作を行うエリア。

| 要素名 | 種類 | 説明 | 備考 |
| :--- | :--- | :--- | :--- |
| **リストエリア** | コンテナ | 追加された仕込み品をカードまたはリスト形式で表示する。 | 初期：空 |
| **(リスト表示項目)** | 要素 | 追加された仕込み品名、量、単位などを表示。 |  |
| **編集アイコン** | アイコン | リスト内の量を再編集する。仕込み画面と同様の機能。 |  |
| **削除アイコン** | アイコン | リストからその行を除外する。仕込み画面と同様の機能。 |  |
| **お品名** | テキスト入力 | 作成するお品に名前を付ける（例：定食、ランチセット等）。 | 必須。初期：空 |
| **合計金額** | テキスト | リストに含まれる全ての仕込み品の値段の合計を表示。 | 初期：¥ 0、自動計算 |
| **お品確定** | ボタン | 全ての入力を確定し、保存処理（確認モーダル）へ進む。 |  |

## 3. 画面イベント
### 3.1 仕込み検索・サジェスト
- ユーザーが「仕込み（検索）」にテキスト入力中、バックエンドAPIを呼び出し、属性が「2」（仕込み品）のものに限定してサジェストを表示する。

### 3.2 値段の算出
- 「量」の値が「半角数字」で入力・変更された際、選択された仕込み品の単価を用いて値段を動的に算出表示する。
- **計算式**: `値段 = 選択仕込み品の単価 * 入力された量`

### 3.3 リストへの追加・編集・削除（共通）
- **追加**: 「リストに追加」等のボタンよりバリデーションを経て右側リストへ反映する。
- **編集**: リスト内の「編集アイコン」を押下すると、その情報が左カラムの入力フォームへ戻り再編集可能となる（仕込み画面の仕様と同等）。
- **削除**: リスト内の「削除アイコン」を押下すると、即座にリストから除外される（仕込み画面の仕様と同等）。

### 3.4 お品の確定・保存処理
- 「お品確定」ボタンが押下されると、以下の順序で処理を行う。
    1. **確認モーダル**: お品確定ボタン押下後、登録内容の確認モーダルを表示する。
    2. **画面遷移**: 確認モーダルで確定後、一覧ページに遷移する。
    3. **タブ選択状態**: 遷移先の一覧ページでは、「お品」のタブが選択された状態で表示するように制御する。

## 4. 画面レイアウト構成（イメージ）
画面は左カラムと右カラムの2カラム構成となる（仕込み画面ベース）。

```text
+---------------------------------------------------------------+
| [Header]                                                      |
+---------------------------------------------------------------+
|                                                               |
|  [左カラム: 仕込み検索・量入力]          [右カラム: リスト・お品入力]   |
|                                                               |
|                                      +-----------------------+ |
|  [仕込み(検索): _______ v]           | リストエリア             | |
|                                      | +-------------------+ | |
|  [量: _______] [ g  ]                | | 仕込み品 1     [E][D]| | |
|                                      | +-------------------+ | |
|  値段: ¥ 120                         | | 仕込み品 2     [E][D]| | |
|                                      | +-------------------+ | |
|  [ リストに追加 ]                      +-----------------------+ |
|                                                               |
|                                      [ 下部設定エリア ]           |
|                                      +-----------+-----------+ |
|                                      | (左)      | (右)      | |
|                                      | お品名    | 合計金額  | |
|                                      | [____]    | ¥ 160     | |
|                                      |           |           | |
|                                      |           | [お品確定]| |
|                                      +-----------+-----------+ |
+---------------------------------------------------------------+
```
