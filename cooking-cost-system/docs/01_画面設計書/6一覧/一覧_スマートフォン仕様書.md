# 一覧 スマートフォン仕様書

## 1. 概要

食材・仕込み・お品の一覧を表示する画面。スマートフォンではテーブルの横幅超過を防ぐため、
カード型リストに変更する。タブ切り替えはそのまま維持する。

| 項目 | 内容 |
|---|---|
| 対象ブレークポイント | `max-width: 640px` |
| デスクトップとの差異 | テーブル → カード型リスト（タブ共通） |

---

## 2. 画面要素

| 要素名 | 種類 | 説明 | 備考 |
|---|---|---|---|
| **ヘッダー** | 共通 | ハンバーガーメニュー付き | |
| **タブ切り替え** | タブバー | 食材 / 仕込み / お品 | 幅100%均等配置 |
| **件数 + 検索バー** | 行 | 件数左・検索右 | |
| **カードリスト** | リスト | タブごとにカード内容が変わる | |

### 食材タブ カード内容

| 項目 | 表示内容 |
|---|---|
| メイン | 食材名（太字） |
| サブ1 | `¥ X.XX / 単位` |
| サブ2 | 購入先 ・ ジャンル |
| 操作 | ✏️ 編集 ・ 🗑️ 削除 |

### 仕込みタブ カード内容

| 項目 | 表示内容 |
|---|---|
| メイン | 仕込み名（太字） |
| サブ1 | `¥ X.XX / g` |
| サブ2 | 使用食材（カンマ区切り・省略あり） |
| 操作 | ✏️ 編集 ・ 🗑️ 削除 |

### お品タブ カード内容

| 項目 | 表示内容 |
|---|---|
| メイン | お品名（太字）+ 原価率バッジ |
| サブ1 | 原価 `¥ XXX` ・ 売価 `¥ XXX` |
| サブ2 | 使用仕込み（カンマ区切り） |
| 操作 | ✏️ 編集 ・ 🗑️ 削除 |

## 4. デスクトップとの変更点

| 項目 | デスクトップ | スマートフォン |
|---|---|---|
| リスト形式 | テーブル（複数列） | カード型リスト（縦積み） |
| 操作ボタン | テーブル列内 | カード右端 44px |
| タブ | ヘッダー下 | 同位置（幅100%均等） |
