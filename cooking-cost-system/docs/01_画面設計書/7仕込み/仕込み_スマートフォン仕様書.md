# 仕込み スマートフォン仕様書

## 1. 概要

仕込み品を作成する画面。デスクトップ版は左右2カラム構成だが、
スマートフォンでは上下の縦1カラムに変更する。
食材選択エリアを上部、構成リストと確定エリアを下部に配置する。

| 項目 | 内容 |
|---|---|
| 対象ブレークポイント | `max-width: 640px` |
| デスクトップとの差異 | 左右2カラム → 上下1カラム |

---

## 2. 画面要素

デスクトップ版と同一要素。配置のみ変更。

## 4. デスクトップとの変更点

| 項目 | デスクトップ | スマートフォン |
|---|---|---|
| レイアウト | 左右2カラム | 上下縦1カラム |
| 食材選択エリア | 左カラム | 上部セクション |
| 構成リスト | 右カラム上部 | 中央セクション |
| 確定エリア | 右カラム下部 | 下部セクション |
| カード表示 | グリッド | 縦積みリスト |
