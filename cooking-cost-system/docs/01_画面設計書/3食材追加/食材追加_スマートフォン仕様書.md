# 食材追加 スマートフォン仕様書

## 1. 概要

食材を新規登録する画面。スマートフォンでは縦1カラムのフォームレイアウトに変更する。
デスクトップ版と機能は同一。

| 項目 | 内容 |
|---|---|
| 対象ブレークポイント | `max-width: 640px` |
| デスクトップとの差異 | フォームを縦1カラムに変更、入力欄の幅を画面幅に合わせる |

---

## 2. 画面要素

| 要素名 | 種類 | 説明 | 備考 |
|---|---|---|---|
| **ヘッダー** | 共通 | ハンバーガーメニュー付き | |
| **画面タイトル** | テキスト | 「食材追加」 | |
| **食材名** | テキスト入力 | 必須 | 幅100% |
| **価格** | 数値入力 | 必須・0以上 | 幅100% |
| **数量** | 数値入力 | 必須・0より大きい | 幅100% |
| **単位** | セレクト/テキスト | g / ml / 個 など | 幅100% |
| **購入先** | Creatable Select | 既存から選択 or 新規作成 | 幅100% |
| **ジャンル** | Creatable Select | 既存から選択 or 新規作成 | 幅100% |
| **バーコードスキャン** | ボタン | カメラ起動 | 幅100% |
| **登録ボタン** | ボタン | 送信 | 幅100%・高さ 48px |

## 4. デスクトップとの変更点

| 項目 | デスクトップ | スマートフォン |
|---|---|---|
| フォームレイアウト | 左右分割あり | 縦1カラム |
| 入力欄幅 | 固定幅 | 画面幅 100% |
| 登録ボタン | 中央配置 | 幅100%・固定高さ 48px |
