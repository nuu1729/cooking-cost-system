# ホーム画面 スマートフォン仕様書

## 1. 概要

ログイン後に表示されるホーム画面。スマートフォンでは縦1カラムのカードグリッドに変更し、
各機能へのアクセスをタップで行えるようにする。

| 項目 | 内容 |
|---|---|
| 対象ブレークポイント | `max-width: 640px` |
| デスクトップとの差異 | カードを縦1〜2カラムに変更、ナビゲーションはハンバーガーメニューへ |

---

## 2. 画面要素

| 要素名 | 種類 | 説明 | 備考 |
|---|---|---|---|
| **ヘッダー** | 共通コンポーネント | ハンバーガーメニュー付きヘッダー | 共通仕様書参照 |
| **ウェルカムメッセージ** | テキスト | ユーザー名を含む挨拶文 | 左寄せ |
| **機能カードグリッド** | グリッドコンテナ | 各機能へのショートカットカード | 2カラム |
| **食材追加カード** | カード | 食材追加画面へ遷移 | |
| **仕込みカード** | カード | 仕込み画面へ遷移 | |
| **お品カード** | カード | お品画面へ遷移 | |
| **一覧カード** | カード | 一覧画面へ遷移 | |
| **メモエリア** | テキストエリア | 自由記述メモ（既存機能） | |

## 4. デスクトップとの変更点

| 項目 | デスクトップ | スマートフォン |
|---|---|---|
| カードレイアウト | 3〜4カラム | 2カラム |
| ナビゲーション | ヘッダーリンク | ハンバーガーメニュー |
| カードサイズ | 固定幅 | 画面幅 50% |
