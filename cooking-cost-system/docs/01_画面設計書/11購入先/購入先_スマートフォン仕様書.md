# 購入先管理画面 スマートフォン仕様書

## 1. 概要

購入先（仕入れ先）マスタを管理する画面。スマートフォンではデスクトップの左右2カラムを
縦積みに変更し、フォームエリア → 一覧エリアの順で表示する。

| 項目 | 内容 |
|---|---|
| 対象ブレークポイント | `max-width: 640px` |
| デスクトップとの差異 | 左右2カラム → 縦積み1カラム（フォーム上・一覧下） |

---

## 2. 画面要素

| 要素名 | 種類 | 説明 | 備考 |
|---|---|---|---|
| **ヘッダー** | 共通コンポーネント | ハンバーガーメニュー付きヘッダー | 共通仕様書参照 |
| **ページタイトル** | テキスト | 「購入先」 | |
| **購入先名入力** | テキスト入力 | 購入先の名称を入力 | 全幅 |
| **編集中表示** | テキスト | 編集モード時に購入先名・使用食材数を表示 | 青背景 |
| **新規ボタン** | ボタン | フォームをリセットして新規登録モードへ | |
| **登録/更新ボタン** | ボタン | 新規時「登録」・編集時「更新」 | 青系・名前未入力時は非活性 |
| **削除ボタン** | ボタン | 編集モード時のみ表示 | 赤系 |
| **エラーメッセージ** | テキスト | 赤色で表示 | |
| **成功メッセージ** | テキスト | 緑色で表示 | |
| **件数バッジ** | テキスト | 登録済み購入先の総件数 | |
| **購入先リスト** | リスト | 順位・名称・食材使用件数。タップで編集モードへ | 全幅 |

---

## 3. レイアウト

```text
+--------------------------------------+
| [🍽 料理原価計算システム]          [≡] |
+--------------------------------------+
|  購入先                              |
|                                      |
|  ── 入力フォーム ──                  |
|  購入先名                            |
|  [______________________________]    |
|                                      |
|  （編集中：業務スーパー / 食材 3件）  |
|                                      |
|  [  新規  ]  [  登録  ]  [  削除  ]  |
|                                      |
|  (エラー or 成功メッセージ)           |
|                                      |
|  ── 購入先一覧 ── (12件)             |
|  使用食材数が多い順                   |
|                                      |
|  +----------------------------------+|
|  | 1  業務スーパー          10件  > ||
|  | 2  コスモス               7件  > ||
|  | 3  ドラッグストア          3件  > ||
|  | ...                             ||
|  +----------------------------------+|
|                                      |
+--------------------------------------+
```

---

## 4. デスクトップとの変更点

| 項目 | デスクトップ | スマートフォン |
|---|---|---|
| レイアウト | 左右2カラム | 縦積み1カラム（フォーム→一覧の順） |
| 入力フィールド | 自動幅 | 全幅 |
| ボタン配置 | 横並び | 横並び（3ボタン均等幅） |
| 一覧行 | テーブル形式 | リスト形式・各行にタップ矢印表示 |
