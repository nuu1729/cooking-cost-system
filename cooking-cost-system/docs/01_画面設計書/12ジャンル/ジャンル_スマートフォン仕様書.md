# ジャンル管理画面 スマートフォン仕様書

## 1. 概要

食材ジャンルのマスタ管理（登録・編集・削除）を行う画面。
スマートフォンではデスクトップの左右2カラムを縦積みに変更し、
フォームエリア → 一覧エリアの順で表示する。

| 項目 | 内容 |
|---|---|
| 対象ブレークポイント | `max-width: 640px` |
| デスクトップとの差異 | 左右2カラム → 縦積み1カラム（フォーム上・一覧下） |

---

## 2. 画面要素

| 要素名 | 種類 | 説明 | 備考 |
|---|---|---|---|
| **ヘッダー** | 共通コンポーネント | ハンバーガーメニュー付きヘッダー | 共通仕様書参照 |
| **ページタイトル** | テキスト | 「ジャンル管理」 | |
| **ジャンル名入力** | テキスト入力 | ジャンル名を入力（最大50文字） | 全幅 |
| **登録するボタン** | ボタン | 新規登録モード時に表示 | 全幅・青系 |
| **更新するボタン** | ボタン | 編集モード時に表示 | 青系 |
| **削除するボタン** | ボタン | 編集モード・食材0件時のみ活性 | 赤系 |
| **キャンセルボタン** | ボタン | 編集モードを解除して新規モードに戻る | |
| **エラートースト** | 通知 | 同名ジャンル登録時（409）に表示 | |
| **削除確認モーダル** | モーダル | 削除前に確認ダイアログを表示 | |
| **件数バッジ** | テキスト | 登録済みジャンルの総件数 | |
| **ジャンルリスト** | リスト | 順位・ジャンル名・食材数・バッジ。タップで編集モードへ | 全幅 |

## 3. レイアウト

```text
+--------------------------------------+
| [🍽 料理原価計算システム]          [≡] |
+--------------------------------------+
|  ジャンル管理                         |
|                                      |
|  ── 入力フォーム ──                  |
|  ジャンル名                          |
|  [______________________________]    |
|                                      |
|  [  登録する  ]                       |
|                                      |
|  （編集モード時）                     |
|  [  更新する  ]  [  削除する  ]       |
|  [  キャンセル  ]                     |
|                                      |
|  ── ジャンル一覧 ── (8件)            |
|  食材使用数の多い順                   |
|                                      |
|  +----------------------------------+|
|  | 1  肉類     [使用中]     5件  >  ||
|  | 2  野菜     [使用中]     3件  >  ||
|  | 3  調味料   [選択中]     2件  >  ||
|  | 4  魚介類               0件  >  ||
|  | ...                             ||
|  +----------------------------------+|
|                                      |
+--------------------------------------+

【削除確認モーダル】
+--------------------------------------+
|  +----------------------------------+|
|  |  ジャンル「肉類」を削除しますか？ ||
|  |                                  ||
|  |  [キャンセル]  [削除する]         ||
|  +----------------------------------+|
+--------------------------------------+
```

---

## 4. デスクトップとの変更点

| 項目 | デスクトップ | スマートフォン |
|---|---|---|
| レイアウト | 左右2カラム | 縦積み1カラム（フォーム→一覧の順） |
| 入力フィールド | 自動幅 | 全幅 |
| 登録ボタン | 自動幅 | 全幅 |
| 更新/削除ボタン | 横並び | 横並び（均等幅） |
| 一覧行 | テーブル形式（# / ジャンル名 / 食材数） | リスト形式・バッジ付き・タップ矢印表示 |
