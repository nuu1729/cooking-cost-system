# アカウント作成画面 スマートフォン仕様書

## 1. 概要

新規ユーザーがアカウントを登録する画面。スマートフォンでは画面幅いっぱいの縦積みフォームに変更し、
タップ操作しやすいボタンサイズを確保する。

| 項目 | 内容 |
|---|---|
| 対象ブレークポイント | `max-width: 640px` |
| デスクトップとの差異 | フォームを全幅表示、入力フィールドとボタンを縦積みに変更 |

---

## 2. 画面要素

| 要素名 | 種類 | 説明 | 備考 |
|---|---|---|---|
| **ロゴ/タイトル** | テキスト | 「料理原価計算システム」 | 中央揃え |
| **メールアドレス入力** | テキスト入力 | メールアドレスを入力 | 全幅、`type="email"` |
| **パスワード入力** | パスワード入力 | パスワードを入力 | 全幅 |
| **パスワード条件表示** | テキスト | 8文字以上・大文字・小文字・数字の条件 | 小字で表示 |
| **エラーメッセージ** | テキスト | バリデーション・登録失敗エラー | 赤色表示 |
| **アカウント作成ボタン** | ボタン | 登録処理を実行 | 全幅・高さ 48px |
| **ログイン画面リンク** | テキストリンク | ログイン画面へ遷移 | 中央揃え |

---

## 4. デスクトップとの変更点

| 項目 | デスクトップ | スマートフォン |
|---|---|---|
| フォーム幅 | 固定幅（中央寄せカード） | 画面幅 100%（左右 padding 16px） |
| ボタンサイズ | 自動幅 | 全幅・高さ 48px |
| タイトル位置 | 上部 or 左 | 中央揃え |
