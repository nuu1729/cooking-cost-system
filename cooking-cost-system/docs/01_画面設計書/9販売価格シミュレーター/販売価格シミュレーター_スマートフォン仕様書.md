# 販売価格シミュレーター スマートフォン仕様書

## 1. 概要

登録済みのお品を選択して販売価格の適正度を診断・シミュレーションする画面。
スマートフォンではデスクトップの左右2カラムを縦積みに変更し、
入力エリア → 結果エリアの順で表示する。

| 項目 | 内容 |
|---|---|
| 対象ブレークポイント | `max-width: 640px` |
| デスクトップとの差異 | 左右2カラム → 縦積み1カラム、指標カードを2×2グリッドに変更 |

---

## 2. 画面要素

| 要素名 | 種類 | 説明 | 備考 |
|---|---|---|---|
| **ヘッダー** | 共通コンポーネント | ハンバーガーメニュー付きヘッダー | 共通仕様書参照 |
| **ページタイトル** | テキスト | 「販売価格シミュレーター」 | アイコン付き |
| **お品選択** | セレクトボックス | 登録済みのお品一覧から選択 | 全幅 |
| **食材原価** | 表示（読み取り専用） | 選択したお品の食材原価 | 全幅 |
| **販売価格入力** | 数値入力 | シミュレーション対象の販売価格 | 全幅 |
| **目標原価率スライダー** | スライダー | 20〜50%の範囲で設定 | 全幅 |
| **推奨販売価格** | 表示 | 目標原価率に基づく逆算価格 | |
| **判定バッジ** | 表示 | ◎/○/△/✗ と判定名 | 大きく中央揃え |
| **主要指標カード** | 2×2グリッド | 原価率・粗利益・粗利益率・FL比率 | 各カードに数値表示 |
| **三分法内訳** | 構成バー | 三分法による内訳表示 | 全幅 |
| **価格帯シミュレーション** | テーブル | 複数価格帯の指標一覧 | 全幅・スクロール対応 |
| **コスト上昇シミュレーション** | リスト | 原価上昇時の影響一覧 | 全幅 |

## 4. デスクトップとの変更点

| 項目 | デスクトップ | スマートフォン |
|---|---|---|
| レイアウト | 左右2カラム | 縦積み1カラム（入力→結果の順） |
| 指標カード | 横並び4枚 | 2×2グリッド |
| 価格帯テーブル | 全カラム表示 | 横スクロール対応 |
| 入力フィールド | 自動幅 | 全幅 |
| 判定バッジ | 右カラム上部 | 入力エリア直下に表示 |
