# アカウント編集画面 スマートフォン仕様書

## 1. 概要

ログイン中のユーザーが自身のアカウント情報を閲覧・編集する画面。
スマートフォンでは3つのセクション（アイコン・基本情報・パスワード変更）を縦積みで表示する。

| 項目 | 内容 |
|---|---|
| 対象ブレークポイント | `max-width: 640px` |
| デスクトップとの差異 | セクションを縦積みに変更、入力フィールドを全幅表示 |

---

## 2. 画面要素

| 要素名 | 種類 | 説明 | 備考 |
|---|---|---|---|
| **ヘッダー** | 共通コンポーネント | ハンバーガーメニュー付きヘッダー | 共通仕様書参照 |
| **ページタイトル** | テキスト | 「アカウント情報」 | |
| **アイコンプレビュー** | 円形画像 | アカウントアイコンのプレビュー（直径 80px） | 中央揃え |
| **アイコンアップロードボタン** | ボタン | タップでファイルダイアログを開く | 全幅 |
| **アイコン削除ボタン** | ボタン | アイコン登録時のみ表示 | 全幅・赤系 |
| **アカウント名入力** | テキスト入力 | アカウント名（任意） | 全幅 |
| **メールアドレス入力** | テキスト入力 | メールアドレス（必須） | 全幅 |
| **現在のパスワード入力** | パスワード入力 | パスワード変更時のみ必須 | 全幅・目玉ボタン付き |
| **新しいパスワード入力** | パスワード入力 | パスワード変更時のみ必須 | 全幅・目玉ボタン付き |
| **新しいパスワード（確認）入力** | パスワード入力 | パスワード変更時のみ必須 | 全幅・目玉ボタン付き |
| **保存するボタン** | ボタン | 全セクションを一括保存 | 全幅・高さ 48px |

## 4. デスクトップとの変更点

| 項目 | デスクトップ | スマートフォン |
|---|---|---|
| レイアウト | カード形式（中央寄せ固定幅） | 縦積み全幅（左右 padding 16px） |
| アイコンプレビュー | 直径 100px | 直径 80px・中央揃え |
| アップロード方法 | ドラッグ＆ドロップ or クリック | タップ（ファイルダイアログ）のみ |
| ボタン | 自動幅 | 全幅・高さ 48px |
