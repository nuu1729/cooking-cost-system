# 食材検索 スマートフォン仕様書

## 1. 概要

登録済み食材を検索する画面。スマートフォンではテーブル形式からカード型リストに変更し、
タップ操作を最適化する。

| 項目 | 内容 |
|---|---|
| 対象ブレークポイント | `max-width: 640px` |
| デスクトップとの差異 | テーブル → カード型リスト、フィルター欄を折りたたみ式に変更 |

---

## 2. 画面要素

| 要素名 | 種類 | 説明 | 備考 |
|---|---|---|---|
| **ヘッダー** | 共通 | ハンバーガーメニュー付き | |
| **検索バー** | テキスト入力 | インクリメンタル検索（300ms debounce） | 幅100% |
| **件数表示** | テキスト | 「全 N 件」 | |
| **食材カード** | カードリスト | 1件につき1カード | |
| *(カード内) 食材名* | テキスト | メインテキスト・太字 | |
| *(カード内) 単価* | テキスト | `¥ X.XX / 単位` | |
| *(カード内) 購入先* | テキスト | サブテキスト | |
| *(カード内) ジャンル* | バッジ | ジャンル名 | |
| *(カード内) 編集* | アイコンボタン | ✏️ | 44×44px |
| *(カード内) 削除* | アイコンボタン | 🗑️ | 44×44px |

## 4. デスクトップとの変更点

| 項目 | デスクトップ | スマートフォン |
|---|---|---|
| リスト形式 | テーブル（列: 名前・単価・購入先・ジャンル・操作） | カード型リスト |
| 操作ボタン | テーブル内小アイコン | カード内 44px ボタン |
