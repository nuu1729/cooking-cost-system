# 共通ヘッダー スマートフォン仕様書

## 1. 概要

スマートフォン（幅 640px 以下）では、デスクトップ版のナビゲーションリンクを
ハンバーガーメニューに置き換え、縦画面での操作性を確保する。

| 項目 | 内容 |
|---|---|
| 対象ブレークポイント | `max-width: 640px`（Tailwind `sm` 未満） |
| デスクトップとの差異 | ナビゲーションリンクを非表示、ハンバーガーボタンを表示 |

---

## 2. 画面要素

| 要素名 | 種類 | 説明 | 備考 |
|---|---|---|---|
| **ロゴ/タイトル** | テキスト | 「料理原価計算システム」またはアイコン | 左寄せ |
| **ハンバーガーボタン** | アイコンボタン | ≡（三本線）アイコン、タップでドロワーを開く | 右端配置・44×44px |
| **ドロワーオーバーレイ** | 半透明レイヤー | ドロワー展開時に背景を暗くする | タップで閉じる |
| **ドロワーメニュー** | スライドパネル | 右側からスライドインするナビゲーションパネル | 幅 280px |

### ドロワーメニュー内の項目

| 項目 | アイコン | 遷移先 |
|---|---|---|
| ホーム | 🏠 | `/` |
| 食材追加 | ➕ | `/ingredients/add` |
| 仕込み | 🔥 | `/dishes/prep` |
| お品 | 🍽 | `/dishes/large` |
| 一覧 | 📋 | `/list` |
| アカウント | 👤 | `/account` |
| ログアウト | 🚪 | ログアウト処理 |

---

## 3. 画面イベント

### 3.1 ハンバーガーボタン押下
1. ドロワーメニューが右からスライドインする（transition: 300ms）
2. 背景にオーバーレイ（黒・透明度 50%）が表示される

### 3.2 ドロワー閉じる
- オーバーレイタップ、または ✕ ボタン押下でドロワーが閉じる
- ドロワー内の項目を選択すると遷移後に自動で閉じる

---
