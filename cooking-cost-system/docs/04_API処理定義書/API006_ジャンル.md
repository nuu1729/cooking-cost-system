# API006 ジャンルマスタ API 処理定義書

## 1. 概要

食材のジャンルをユーザーごとに管理するマスタ API。ジャンルは `items.genre_id` でFKとして参照される。

---

## 2. エンドポイント一覧

| # | メソッド | パス | 処理概要 |
|---|---------|------|---------|
| 1 | GET | /api/genres | ジャンル一覧取得（使用数降順） |
| 2 | POST | /api/genres | ジャンル新規登録 |
| 3 | PUT | /api/genres/{id} | ジャンル名更新 |
| 4 | DELETE | /api/genres/{id} | ジャンル削除 |

全エンドポイント：JWT認証必須（`g.user_id` でユーザー分離）

---

## 3. 各エンドポイント詳細

### 3.1 GET /api/genres

**処理フロー**

1. `Genre.query.filter_by(user_id=g.user_id)` で全ジャンルを取得
2. 各ジャンルに `Item.query.filter_by(user_id, item_type=1, genre_id=genre.id).count()` で食材数を付加
3. 食材数降順でソートして返却

**レスポンス**

```json
{
  "success": true,
  "data": [
    { "id": 1, "name": "肉類", "ingredient_count": 5, ... },
    { "id": 2, "name": "野菜", "ingredient_count": 3, ... }
  ],
  "count": 7
}
```

---

### 3.2 POST /api/genres

**リクエストボディ**

```json
{ "name": "乳製品" }
```

**処理フロー**

1. `name` が空の場合 → 400 VALIDATION_ERROR
2. `Genre.query.filter_by(user_id, name)` で重複チェック → 重複時 409 CONFLICT
3. `Genre(user_id, name)` を生成して INSERT
4. `ingredient_count: 0` を付加して返却（201）

---

### 3.3 PUT /api/genres/{id}

**リクエストボディ**

```json
{ "name": "魚介類" }
```

**処理フロー**

1. 対象ジャンルが存在しない場合 → 404
2. `name` が空の場合 → 400 VALIDATION_ERROR
3. 同名の別ジャンルが存在する場合 → 409 CONFLICT
4. `genre.name` を更新
5. `Item.query.filter_by(user_id, genre=old_name).update({'genre': name})` で食材の genre テキスト列も同期
6. 更新後のジャンルオブジェクト（ingredient_count 付き）を返却

---

### 3.4 DELETE /api/genres/{id}

**処理フロー**

1. 対象ジャンルが存在しない場合 → 404
2. `Item.query.filter_by(user_id, item_type=1, genre_id=id).count() > 0` → 409 CONFLICT（使用中保護）
3. `db.session.delete(genre)` で削除（ON DELETE SET NULL により items.genre_id は NULL になる）

## 4. items テーブルとの連携

### 食材登録時（POST /api/ingredients）

```python
if genre_id is not None:
    genre_obj = Genre.query.filter_by(id=genre_id, user_id=g.user_id).first()
    if genre_obj:
        genre = genre_obj.name       # genre テキスト列も同期
        resolved_genre_id = genre_obj.id
```

### 食材更新時（PUT /api/ingredients/{id}）

```python
if 'genre_id' in body and body['genre_id'] is not None:
    genre_obj = Genre.query.filter_by(id=body['genre_id'], user_id=g.user_id).first()
    if genre_obj:
        item.genre_id = genre_obj.id
        item.genre = genre_obj.name
```

## 5. エラーコード

| HTTPステータス | コード | メッセージ例 |
|--------------|-------|------------|
| 400 | VALIDATION_ERROR | name は必須です |
| 404 | NOT_FOUND | ジャンルが見つかりません |
| 409 | CONFLICT | このジャンルは N 件の食材で使用中のため削除できません |
| 409 | CONFLICT | 「〇〇」はすでに登録されています |
